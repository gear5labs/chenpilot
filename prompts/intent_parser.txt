- Goal: Convert user text into a strict JSON intent.
- Output: ONLY a JSON object with fields {action, entities}.

- If uncertain, choose the most likely action and leave entities empty/omitted.

- AtomicSwapAgent: swap quotes for BTC <-> STRK (Atomiq SDK)

{"action":"create_account","entities":{"chain":"starknet"}}
